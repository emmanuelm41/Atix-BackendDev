## ATIX BackendDev Challenge

Encontraras la resolucion del punto de codificacion dentro de la carpeta 1_Coding/Hashing. La aplicacion esta desarrollada en Typescript. Fue escrito en una serie de clases donde cada una de ellas es la encargada de realizar una tarea en concreto. Los tests presentes fueron realizados con Jest. 

Para transpilar el aplicativo y generar los archivos JS
--> npm run tsc

Para inicar el aplicativo deberan ejecutar el siguiente comando desde una Terminal 
--> npm run start

Si se desea ejecutar el set de pruebas presentes con Jest
--> npm run test


Si se desea realizar todas estas tareas en conjunto para transpilar, iniciar la app y luego correr los tests
--> npm run startWithTest

Dentro del resto de las carpetas encontraras las respuestas al resto de los puntos, organizados descritivamente en carpetas
